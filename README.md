# 六账户框架 - 个人资本系统搭建助手

> 基于太阳李博良六账户方法论，帮助普通人系统化构建长期生活底气

[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

## 背景

外部经济环境已发生结构性变化，普通人不能再依赖单一收入来源或被动等待机会。六账户框架主张将自身视为一个需要持续经营和资本积累的系统，主动构建六个相互关联的个人账户，以形成长期的安全感和选择权。

## 六个账户

六个账户形成「防护层 → 增长层 → 杠杆层 → 元认知层」的递进结构：

### 🛡️ 防护层

**现金账户** — 守住财富，抵御风险。

在经济不确定性增大的环境下，普通人最大的财务风险不是回报率不够高，而是本金永久性损失。现金账户强调防御性策略：在不确定时期，持有流动性本身就是一种收益。

- 计算 6 个月基本生活开销作为目标储备
- 设定每月储蓄比例
- 列出并削减不必要的支出

没有现金安全垫，其他账户的长期投入都无从谈起。

### 📈 增长层

**技能账户** — 持续自我增值，可迁移能力的复利积累。

以固定节奏持续投入时间提升技能，将技能提升从"随缘学习"升级为"定期定投"。关键在于持续性和规律性，而非单次时长。

推荐技能方向：表达、写作、AI 使用、数据分析、销售、短视频剪辑、英语、行业理解。这些技能的共同特点：高可迁移性（不绑定特定公司）、高杠杆性（小投入大回报）、与 AI 时代相关。

**作品账户** — 能力可视化，让结果替你说话。

持续产出并公开你的作品，用可见的成果代替自我陈述来证明能力。在信息过载的时代，"我很厉害"毫无意义，"这是我做的东西"才是唯一有效的信用媒介。

作品形式：文章、视频、代码仓库、项目复盘、课程、模板、报告。作品是求职、自由职业、创业场景中最强的谈判筹码——它不依赖任何中间人背书，自带证明力。

### 🔗 杠杆层

**信用账户** — 社会信任资本，兑现承诺建立声誉。

信守承诺，保持长期真实有效的输出。信用不是一次性的，而是在一次次兑现中积累的复利。设定小而确定的承诺并 100% 完成，持续输出有价值的内容，积累他人的推荐和评价。

**关系账户** — 高质量协作网络，合作杠杆与人脉复利。

建立互利的高质量合作关系。关键是先提供帮助再寻求帮助——关系的复利在于你为他人创造了多少价值。每周主动联系一位有价值的人，建立互利的合作机制。

### 🧠 元认知层

**赛博日记账户** — AI 辅助的经验复利系统，贯穿前五个账户的元账户。

每天花 5 分钟记录：做了什么、学了什么、遇到什么问题。每周用 AI 做一次成长复盘，将经验提炼为可复用的方法论。赛博日记让 AI 帮你从零散经验中提炼智慧，实现个人成长的复利效应。

## 使用流程

### 第一步：账户诊断

对六个账户逐一打分（1-10 分）：

```
1. 现金账户：你有多个月的应急储备？
2. 技能账户：过去三个月学了什么新东西？
3. 作品账户：有什么可以展示给别人看的成果？
4. 信用账户：有多少人愿意为你推荐或背书？
5. 关系账户：你能列出几个高质量的合作伙伴？
6. 赛博日记账户：你用什么方式记录和复盘成长？
```

### 第二步：制定行动计划

根据诊断结果，优先投入最薄弱的账户。每个账户都有对应的行动方案模板。

### 第三步：定期复盘

使用以下模板进行每周/每月复盘：

```markdown
# 成长复盘 YYYY-MM-DD

## 现金账户
- 当前储蓄进度：___/目标___

## 技能账户
- 本周投入学习时间：___小时
- 新掌握的技能：___

## 作品账户
- 本周产出的作品：___

## 信用账户
- 兑现的承诺：___

## 关系账户
- 新建立的联系：___

## 赛博日记账户
- 本周最重要的一个教训：___
- AI 复盘发现：___
```

## 底层思维模型

框架的共同底层逻辑：

- **复利思维**：六个账户都通过持续小投入获得长期大回报——每天 1% 的进步，一年后是 37.8 倍
- **反脆弱**：六账户框架本质上是一套增强个人反脆弱性的策略，分散风险、建立多重安全网
- **系统思维**：六个账户不是孤立的，而是相互输入输出的有机系统。技能 → 作品 → 信用 → 关系，现金是底盘，赛博日记是操作系统
