## 简介

一个 AI Agent 技能（Skill），基于抖音创作者「太阳李博良」提出的六账户方法论，帮助普通人系统化构建六个个人资本账户：

| 层级 | 账户 | 功能 |
|------|------|------|
| 防护层 | 现金账户 | 守住财富，抵御风险 |
| 增长层 | 技能账户 | 持续自我增值 |
| 增长层 | 作品账户 | 能力可视化展示 |
| 杠杆层 | 信用账户 | 社会信任资本 |
| 杠杆层 | 关系账户 | 高质量协作网络 |
| 元认知层 | 赛博日记账户 | AI辅助经验复利 |
