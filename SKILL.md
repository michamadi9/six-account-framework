# 六账户框架：个人资本系统搭建助手

> 基于太阳李博良的六账户方法论，帮助普通人系统化构建长期生活底气。

## 触发词

`六个账户`、`个人成长规划`、`六账户诊断`、`人生框架`、`成长复盘`、`怎么提升自己`、`赚虾米`

## 框架核心

六个账户形成「防护层 → 增长层 → 杠杆层 → 元认知层」的递进结构：

| 层级 | 账户 | 核心功能 |
|------|------|----------|
| 防护层 | 现金账户 | 守住财富，抵御风险，提供生存底线 |
| 增长层 | 技能账户 | 持续自我增值，可迁移能力的复利积累 |
| 增长层 | 作品账户 | 能力可视化，让结果替你说话 |
| 杠杆层 | 信用账户 | 社会信任资本，兑现承诺建立声誉 |
| 杠杆层 | 关系账户 | 高质量协作网络，合作杠杆与人脉复利 |
| 元认知层 | 赛博日记账户 | AI辅助的经验复利，贯穿前五个账户 |

## 使用流程

### 第一步：账户诊断

帮助用户评估六个账户的当前状态：

```
请为每个账户打分（1-10分）：
1. 现金账户：你有多少个月的应急储备？
2. 技能账户：过去三个月学了什么新东西？
3. 作品账户：有什么可以展示给别人看的成果？
4. 信用账户：有多少人愿意为你推荐或背书？
5. 关系账户：你能列出几个高质量的合作伙伴？
6. 赛博日记账户：你用什么方式记录和复盘成长？
```

### 第二步：制定行动计划

根据诊断结果，优先投入最薄弱的账户：

**现金账户行动方案：**
- 计算6个月基本生活开销作为目标储备
- 设定每月储蓄比例
- 列出不必要的支出并削减

**技能账户行动方案：**
- 每周固定2-4小时用于技能学习
- 推荐技能方向：表达、写作、AI使用、数据分析、销售、短视频剪辑、英语、行业理解
- 关键是持续性和规律性，而非单次时长

**作品账户行动方案：**
- 本周产出一个可展示的作品（文章/视频/代码/报告/模板）
- 建立作品集（GitHub/个人博客/飞书文档）
- 每次完成项目后写一份复盘

**信用账户行动方案：**
- 信守承诺，设定小而确定的承诺并100%完成
- 持续输出有价值的内容
- 积累他人的推荐和评价

**关系账户行动方案：**
- 每周主动联系一位有价值的人
- 先提供帮助再寻求帮助
- 建立互利的合作机制

**赛博日记账户行动方案：**
- 每天花5分钟记录：做了什么、学了什么、遇到什么问题
- 每周用AI做一次成长复盘
- 将经验提炼为可复用的方法论

### 第三步：定期复盘（每周/每月）

使用以下模板：

```markdown
# 成长复盘 YYYY-MM-DD

## 现金账户
- 当前储蓄进度：___/目标___
- 本月削减的开支：___

## 技能账户
- 本周投入学习时间：___小时
- 新掌握的技能：___

## 作品账户
- 本周产出的作品：___
- 获得的反馈：___

## 信用账户
- 兑现的承诺：___
- 收到的认可：___

## 关系账户
- 新建立的联系：___
- 促成的合作：___

## 赛博日记账户
- 本周最重要的一个教训：___
- AI复盘发现：___
```

## 关键原则

1. **现金先行**：没有现金安全垫，其他账户的长期投入无从谈起
2. **技能是资产**：不绑定特定公司或岗位的能力才是真正的安全感
3. **作品即信用**：说"我很厉害"毫无意义，展示"这是我做的东西"才是唯一有效的证明
4. **复利思维**：六个账户的共同底层逻辑——通过持续小投入获得长期大回报
5. **AI作为杠杆**：赛博日记账户让AI帮你从经验中提炼智慧

## 快速启动

如果你想立刻开始，只需做这三件事：
1. 写下你目前的现金储备（诚实面对）
2. 选一个技能方向，本周投入2小时
3. 今天产出一份小作品（哪怕只有200字）
